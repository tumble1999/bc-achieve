## Usage
```js
createAchievement({
		mod: BCAchieve,
		name: "Test Achivement",
		description: "Test the achivement system."
	});

	createAchievement({
		mod: BCAchieve,
		name: "Test Incremental Achivement",
		description: "Test the achivement system.",
		amount: 5
	});

	createAchievement({
		mod: BCAchieve,
		name: "Test Image Achivement",
		description: "Test the achivement system.",
		icon: "https://boxcritters.com/images/menu/Btn_Menu_Items_A.png"
	});
	```
